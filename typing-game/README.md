# ゆびたんけん！タイピングのたび

小学2年生（タイピング初学者）向けの、指の位置を確実に身につけるタイピングゲーム。

## 設計のねらい

- **指の位置を最優先で習得**：常時キーボードを画面表示。指ごとに色分けし、次に押すキーを大きく強調＋使う指の名前をヒント表示。
- **学校の進度に合わせた段階式**：ホームポジション → あ行 → か行 → さ行 → あ〜さ行の単語 → ポケモン名 → 技名（長め）の7ステージ。
- **苦手キーの重点練習**：ミスが多いキーを記録し、次回出題の冒頭付近に優先配置。
- **続けたくなる工夫**：ステージごとのバッジ、ベスト記録、ポケモン画像、クリア演出。
- **オフライン対応**：単一HTMLファイルとして動作。ポケモン画像はネット経由（PokeAPI）または `images/pokemon/` からのフォールバック。

## 参考にした既存ゲーム

- プレイグラムタイピング（指運用の可視化／苦手キー重点出題）
- Typing Land（段階アンロック／バッジ）
- ポケモンPCトレーニング（低学年向けの達成報酬）
- Nitro Type（ベスト記録の継続意欲）

## 使い方

1. `typing-game/index.html` をブラウザでダブルクリックして開く。
2. **タイトル画面**で「ローマ字をひょうじする」のオン/オフを設定。最初は ON 推奨。
3. **ステージをえらぶ**から開いているステージを選択（最初はステージ1のみ解放）。
4. 表示されたかな・ローマ字を、画面のキーボードに従ってタイプ。正答率がしきい値以上ならクリア＝バッジ獲得＆次ステージ解放。

### ホスティング（任意）

ブラウザでローカルから直接開いても動作しますが、`fetch`系の都合や PokeAPI 画像取得が安定しないことがあれば、簡易サーバで配信してください：

```sh
cd typing-game
python3 -m http.server 8000
# http://localhost:8000 を開く
```

## 進捗の保存

ブラウザの `localStorage` に保存しています。タイトル画面の「きろくをリセット」で初期化できます。
別の端末や別のブラウザでは進捗は引き継がれません。

## ステージ構成

| # | 名前 | 内容 | 合格率 |
|---|------|------|--------|
| 1 | ホームポジション | `asdfjkl;` の連打 | 85% |
| 2 | あ行 | a, i, u, e, o | 85% |
| 3 | か行 | ka, ki, ku, ke, ko | 85% |
| 4 | さ行 | sa, shi, su, se, so | 85% |
| 5 | あ〜さ行のことば | きかい、おかし、こけし、… | 80% |
| 6 | ポケモンずかん | ピカチュウ、ゼニガメ、… | 70% |
| 7 | ポケモンのわざ | でんこうせっか、… | 70% |

## カスタマイズ

- **単語追加**：`js/words.js` の各配列に `{ kana, romaji }` を追加。
- **画像追加**：`images/README.md` 参照。
- **見た目調整**：`css/style.css` を編集。

## 操作

- バックスペースは無効化されています（正しいキーを押すまで進めない設計＝指位置の固定化のため）。
- ステージ中の「← やめる」で中断できます。
