# ゆびたんけん！タイピングのたび

小学2年生（タイピング初学者）向けの、指の位置を確実に身につけるタイピングゲーム。
**訓令式ローマ字**（si, ti, tu, hu, zi）で打鍵数最小化。**音声ファイル不要**（Web Audio API合成）。

## 3つのモード

### 📝 れんしゅう（14ステージ）
段階的にバッジを集めるアンロック式。
1. ホームポジション（asdfjkl;）
2. あ行
3. か行
4. さ行（し=SI）
5. た行（ち=TI、つ=TU）
6. な行
7. は行（ふ=HU）
8. ま行・や行
9. ら行・わ行・ん（ん=NN）
10. だくおん（が・ざ・じ=ZI・だ・ば）
11. はんだくおん・ようおん（ぱ・きゃ・しゃ=SYAなど）
12. いろいろなことば
13. ポケモンずかん（画像つき）
14. ポケモンのわざ（長め）

### ⚔️ ポケモンバトル
赤緑風UIで本格ポケモンバトル。
- **自分のポケモン選択**：ピカチュウ／フシギダネ／ヒトカゲ／ゼニガメ（御三家+ピカ）
- **トレーナー選択**：むしとりしょうねん（弱）→たんぱんこぞう（中）→ジムリーダー ナンジャモ（強）
- **戦闘**：3つの技から選んで技名をタイピング。文字数が多いほどダメージ大、ミスでダメージ減
- **HPバー**：緑→黄→赤と色変化、攻撃で揺れる、倒れる音
- **ターン制**：自分→相手→自分... 相手のAIはランダムに技を選択

ダメージ計算：`max(1, 技の威力 + 文字数*2 - ミス数*3)`

### ⏱️ タイムアタック
60秒で短い単語を次々タイプ。**スコア = 正解文字数*10 - ミス数*3 + 完答単語数*20**

## 音響

すべて **Web Audio API による動的合成**（音声ファイル不要）：
- **打鍵音**：使う指ごとに違う音階（左小指=C5、右小指=C6 のスケール状）
- **ミス音**：低音ノイズバースト
- **ステージクリア音**：上昇アルペジオ
- **失敗音**：下降3音
- **攻撃音**：高→低スイープ＋ノイズ
- **被弾音**：揺らぐ中低音
- **倒れる音**：下降スイープ
- **勝利音**：7音上昇アルペジオ
- **BGM**：メニュー用とバトル用の2種類（軽快ループ）

タイトル画面の「おとを ならす」チェックボックスで一括 ON/OFF 切替可能。

## 設計のねらい

- **指の位置を最優先で習得**：常時キーボードを画面表示。指ごとに色分けし、次に押すキーを大きく強調＋使う指の名前をヒント表示
- **訓令式統一**：打鍵数を最小化（し=si、ち=ti など）
- **苦手キーの自動補強**：ミス率の高いキーを次回出題の冒頭付近に優先配置
- **段階アンロック**：正答率が閾値（85%→70%へ徐々に緩和）に達するとバッジ獲得＆次解放
- **続けたくなる工夫**：バッジ、ベスト記録、ポケモン画像、3モード切替、バトル勝利報酬
- **オフライン対応**：単一HTMLとして動作。ポケモン画像はPokeAPI→`images/pokemon/`の順でフォールバック

## 使い方

### お父さんのPCで開く

```sh
git clone https://github.com/macsa2ki25-eng/SpecialAigent.git
cd SpecialAigent
git checkout claude/typing-game-kids-qO3da
cd typing-game
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く。

`index.html` を直接ダブルクリックでも動きますが、**PokeAPI画像と一部機能はローカルサーバ経由でないと安定しません**。

### 推奨環境

- Chrome / Edge / Safari の最新版
- ノートPC または タブレット＋外付け物理キーボード
- **IMEは必ずOFF（半角英数モード）**
- スピーカーまたはヘッドフォン

## 進捗の保存

ブラウザの `localStorage` に保存（バッジ、ベスト記録、苦手キー、バトル勝利、タイムアタック最高点）。
タイトル画面の「きろくをリセット」で初期化。

## PokeAPIについて

- **画像**：`https://raw.githubusercontent.com/PokeAPI/sprites/...` から取得（前向き／後ろ姿）
- **技データ**：`https://pokeapi.co/api/v2/move/{name}` から日本語名・威力を取得可能（`Pokemon.getMove()` 経由）
- バトルの技セットは応答性のため事前定義（API呼び出しの遅延を回避）

## カスタマイズ

- **単語追加**：`js/words.js` の配列に `{ kana, romaji }` を追加
- **ポケモン追加**：`js/words.js` の `STAGE_POKEMON` に `{ kana, romaji, pokeId, slug }` を追加
- **バトル参加ポケモン追加**：`js/battle.js` の `PLAYER_POKEMONS` に追加（技を3つ定義）
- **トレーナー追加**：`js/battle.js` の `TRAINERS` に追加
- **画像差し替え**：`images/pokemon/<slug>.png` に置くとPokeAPIより優先
- **音色変更**：`js/audio.js` の `FINGER_FREQ` で各指のピッチを調整

## 操作

- バックスペースは無効（指位置の固定化のため、正しいキーを押すまで進めない）
- 各画面の「← もどる」で安全に中断可能
- IMEは必ずOFFで（IMEがONだと変換候補が出てしまう）
