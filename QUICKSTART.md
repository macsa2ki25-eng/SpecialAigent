# SpecialAigent クイックスタートガイド

## 方法A: Claude Code で使う（簡単・推奨）

### やること

1. Claude Code のターミナルで、このフォルダに移動する
2. 命令を打つ
3. 終わり

### 具体的な手順

**ステップ1: このフォルダに入る**

Claude Codeのターミナル（今この画面）で：

```
cd SpecialAigent
```

これだけで、Claude Codeが自動的に「将軍モード」になります。
（CLAUDE.mdを読んで、戦国口調で動き始めます）

**ステップ2: 命令を打つ**

あとは普通にチャットするだけです。例：

```
英語のテストを作って。範囲は関係代名詞で、難易度は標準。
```

将軍（Claude Code）が「承知つかまつった！」と言って、
足軽（サブエージェント）を並列で起動して、テストを作ってくれます。

**ステップ3: 結果を受け取る**

完成したテストは画面に表示され、output/ フォルダにも保存されます。

### 命令の例

| やりたいこと | 命令例 |
|---|---|
| 英語テスト作成 | 「Lesson 3-4の範囲で標準難易度のテストを作って」 |
| ワークシート | 「関係代名詞のワークシートを作って」 |
| 指導案 | 「Lesson 5の指導案を50分で作って」 |
| 志望理由書添削 | 「この志望理由書を添削して：（本文を貼る）」 |
| 進路だより | 「11月号の進路だよりを作って」 |
| 保護者向け通知 | 「保護者会の案内文を作って」 |
| 議事録 | 「職員会議の議事録を作って：（メモを貼る）」 |
| Wi-Fiトラブル | 「3階のWi-Fiが繋がらない原因と対処法を教えて」 |
| マニュアル | 「新任の先生向けにWi-Fi接続マニュアルを作って」 |
| DX提案 | 「出欠管理をデジタル化する方法を提案して」 |
| 自動化 | 「成績一覧のCSVを集計するPythonスクリプトを作って」 |

## 方法B: tmux で使う（見た目が楽しい・上級者向け）

### 必要なもの

- tmux（ターミナルマルチプレクサ）
- WSL2（Windowsの場合）

### 手順

```bash
# 1. WSLを開く（Windowsの場合）
# 2. セットアップ
cd SpecialAigent
bash setup.sh

# 3. tmuxに入る
tmux attach -t multiagent

# 4. 将軍画面（紺色背景）で命令を打つ
# 5. Ctrl+b → 2 で足軽の動きが見える
```

### tmux操作

| キー | 何が起きる |
|---|---|
| `Ctrl+b` → `0` | 将軍画面に切り替え（命令を打つ画面） |
| `Ctrl+b` → `1` | 家老画面（ダッシュボード） |
| `Ctrl+b` → `2` | 足軽画面（3人が動いてる） |
| `Ctrl+b` → `d` | tmuxから離脱（裏で動き続ける） |

## Q&A

**Q: CLAUDE.mdって何ですか？**
A: Claude Codeが自動的に読む設定ファイルです。このファイルがあるフォルダに
   入ると、Claude Codeが将軍モードになって戦国口調で動きます。

**Q: API消費が心配です**
A: Proプラン対応で設計しています。足軽は最大3体、単純な作業は1体だけ使います。
   普段使いで問題ない範囲です。

**Q: 学校のネットワークでも動きますか？**
A: Claude Codeが動く環境なら動きます。今この画面で会話できているなら大丈夫です。

**Q: 生成されたファイルはどこにありますか？**
A: output/ フォルダに保存されます。ファイル名に日付と内容が入っています。
