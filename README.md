# SpecialAigent

Claude Code で動かすX（旧Twitter）収益化自動化エージェント。

## 概要

雑学・ライフハック・心に響く言葉をテーマに、X上で自動的にインプレッションを獲得し、最終的にX Premium収益分配プログラムで収益化することを目標にしたシステム。

**運用方針: 半自動（生成は自動、投稿前に人間が確認）**

## アーキテクチャ

```
┌────────────────────┐   ┌────────────────────┐   ┌────────────────────┐
│ 1. 投稿文生成       │ → │ 2. 承認キュー       │ → │ 3. X APIで投稿      │
│ (Claude APIで)     │   │ (人間が1タップ承認) │   │ (Free tierで投稿)  │
└────────────────────┘   └────────────────────┘   └────────────────────┘
         ↑                                                   │
         │                                                   ↓
┌────────────────────┐                            ┌────────────────────┐
│ バズ型テンプレ       │                            │ 4. 実績記録         │
│ (templates/*.yaml) │                            │ (手動/半自動)      │
└────────────────────┘                            └────────────────────┘
```

## 使い方の要点

```bash
# 1. 投稿を5件生成して承認キューに入れる
python -m src.cli generate --count 5

# 2. 承認キューの内容を確認＆承認
python -m src.cli review

# 3. 承認済みをXに投稿
python -m src.cli publish

# 4. 予約投稿（1日3回、朝昼夜に自動）
python -m src.cli schedule
```

## Claude Code スキル

`.claude/skills/` 配下にあるSKILL.mdを介して、Claude Codeに

- 「今日の投稿を5件生成して」
- 「承認済みの投稿を1件投稿して」
- 「先週の投稿実績をまとめて」

といった自然言語の指示で全工程を実行させられる。
